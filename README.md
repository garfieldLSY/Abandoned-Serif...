# 緣宋體（Garf Serif） 
## 簡介
基于I.明體、IPA明體、馬醉木明體、紫陽花明體以及很多IPA系列的宋體字體修改而來，算是一個究極縫閤怪了。

這玩意不那麽追求字理，我是依照新舊字形對照表、大陸字形、韓明體的字形和康熙字典的字形把N個字體縫閤而來（當然縫閤了之後肯定有漏掉的地方，同志們也别説我糟踐東西）

不過再怎麽改，再怎麽縫合，也得按照規定來，畢竟這字體也是有法可依的。
